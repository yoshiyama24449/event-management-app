# event-management-app

# イベント管理アプリ



### 技術スタック

* Next.js
* FastAPI
* PostgreSQL
* Docker Compose
* Codespaces
* AWS EKS



### アプリ仕様

* 社内向けアプリ
* ログイン必須。
* ユーザはイベント検索、直近のイベント一覧の確認、イベント詳細の確認が可能
* ユーザはイベントに対してイベント参加登録、イベント参加キャンセルができる。参加はしないが、保留や後で確認したい場合はブックマーク機能を使用することができる。
* イベントには定員を設定できる。定員が埋まったイベントが非表示になることはないが、定員を超えて参加申請が通ることはない。定員の設定は必須である。
* イベント詳細表示画面には、コメント欄のようなものがあり、質疑応答や感想の共有に使用されることを想定している。
* ユーザはイベント登録ができる。また自分が登録したイベントに限り編集、削除が可能。
* ユーザはダッシュボードを見ることができる。次の3点を分かりやすく確認できる。
  * 作成したイベント（投稿されたコメントや参加人数と参加者一覧などが確認可能、クリックすることでイベント詳細画面に遷移可能）
  * 投稿したコメントとそれに対する返信
  * 参加予定のイベントやブックマークしたイベント（カレンダービューも存在）
