# event-management-app

# イベント管理アプリ



### 技術スタック

* Next.js
* FastAPI
* PostgreSQL
* Docker Compose
* Codespaces
* AWS EKS



### アプリ仕様

* 社内向けアプリ
* ログイン必須。メールアドレスの騙りや入力ミスを防ぐために、OICDによるシングルサインオンを利用する。利用される社内メールアドレスはMicrosoftアカウントである。
* ユーザはイベント検索、直近のイベント一覧の確認、イベント詳細の確認が可能
* ユーザはイベントに対してイベント参加登録、イベント参加キャンセルができる。参加はしないが、保留や後で確認したい場合はブックマーク機能を使用することができる。
* イベントには定員を設定できる。定員が埋まったイベントが非表示になることはないが、定員を超えて参加申請が通ることはない。定員の設定は必須である。
* イベント詳細表示画面には、コメント欄のようなものがあり、質疑応答や感想の共有に使用されることを想定している。
* 一般ユーザでもイベント登録が可能。また自分が登録したイベントに限り編集、削除が可能。
* ダッシュボードページがあり、作成したイベントがあれば、投稿されたコメントや参加人数と参加者一覧が分かりやすく確認でき、イベント詳細画面に遷移してコメント対応もできる。他には、投稿したコメントとそれに対する返信も一目で確認することができる。さらには、カレンダー形式で参加予定のイベントやブックマークしたイベントを確認することもできる。
